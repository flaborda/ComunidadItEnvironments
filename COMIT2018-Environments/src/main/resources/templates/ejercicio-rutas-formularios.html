Ejercicio: Realizar los siguientes items utilizando todo lo visto en clase de controladores, rutas, templates, formularios, etc.

  1. Hacer una ruta /sorteo que muestre en pantalla un numero entre 0 y 10. El codigo para generar numeros aleatorios esta en Slack
  2. Hacer una ruta /sorteo/{n} que muestre en pantalla un numero entre 0 y n.
  3. Hacer que la ruta / muestre una pagina de bienvenida con dos links (tag a) que apunten a las rutas de los ejercicios anteriores.
  4. Hacer una ruta /completar-datos que mostrar un formulario con campos nombre, apellido, email, y comentario.
  5. Hacer una ruta para procesar ese formulario y si ninguno de los datos esta vacio, mostrar una pantalla de Gracias por contactarnos.
  6. Enganchar ese formulario con esa ruta (si no lo hicieron en el paso anterior)
  7. Hacer que la pantalla de Gracias ademas incluya toda la info recien cargada por el usuario, es decir, nombre, apellido, email y su comentario.
  8. Hacer que si algun dato esta vacio, se vuelva a mostrar el formulario original con los datos pre cargados.
